Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę podzielić podane zdanie na osobne wyrazy, napisać każdy wyraz wielką literą (pierwszy znak tytułu, a pozostałe małą) i połączyć wyrazy z powrotem w zdanie za pomocą białych spacji.

Ok, od czego powinienem zacząć?

Proszę podzielić podane zdanie na osobne słowa za pomocą metody string.split() z argumentem białej spacji. Proszę zastosować określoną funkcję do każdego słowa za pomocą metody array.map(). 

sentence.split(' ').map(word => ...)
Co robić dalej?

Pierwszy znak każdego słowa (charAt()) musi zostać zamieniony na duże litery za pomocą metody string.toUpperCase(). 

word.charAt(0).toUpperCase()
Nadal potrzebuję podpowiedzi.

Resztę słowa (string.slice()) proszę zamienić na małe litery za pomocą toLowerCase(). 

word.slice(1).toLowerCase()

Teraz proszę połączyć części słowa za pomocą +.

Jak połączyć wszystkie razem?

Tak, proszę połączyć wszystkie słowa pisane wielką literą z powrotem w jedno zdanie za pomocą metody string.join() z argumentem białej spacji.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi, zwrócić wynikowe zdanie i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Oto całe rozwiązanie:

function toTitleCase(sentence: string): string {

    return sentence
        .split(' ')
        .map(word => word.charAt(0).toUpperCase() + word.slice(1).toLowerCase())
        .join(' ');
}

Proszę zamienić całe zdanie na małe litery i zastąpić każdą pierwszą literę wielką przy użyciu wyrażeń regularnych.

function toTitleCase(sentence: string): string {

    return sentence.toLowerCase().replace(/\b\w/g, c => c.toUpperCase());
    }
