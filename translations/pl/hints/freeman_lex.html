Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać pomysł 💡! Proszę podzielić podane zdanie na osobne wyrazy, napisać każdy wyraz wielką literą (pierwszy znak tytułu, a pozostałe małą) i połączyć wyrazy z powrotem w zdanie za pomocą białych spacji.

Ok, od czego powinienem zacząć?

Proszę podzielić podane zdanie na osobne słowa za pomocą metody str.split() bez argumentu. Otrzymają Państwo listę słów. 

lst = sentence.split()
Co robić dalej?

Każde słowo powinno zostać przekonwertowane za pomocą metody str.capitalize(). Proszę użyć funkcji map(), aby zaimplementować powyższą metodę do każdego słowa. 

obj = map(str.capitalize, lst)

Map funkcja zwraca obiekt i można go użyć jako argumentu w innej funkcji.

Nadal potrzebuję podpowiedzi.

Teraz musi Pan połączyć wszystkie słowa pisane wielką literą z powrotem w jedno zdanie za pomocą metody str.join() z argumentem białej spacji. 

return " ".join(obj)

Może Pan to zrobić bez dodatkowych zmiennych!

return " ".join(map(str.capitalize, sentence.split()))
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi, zwrócić wynikowe zdanie i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Oto całe rozwiązanie:

def to_title_case(sentence: str) -> str:

    return " ".join(map(str.capitalize, sentence.split()))
