如何连接所有部分？

是的，您需要使用string.join() 方法将所有大写单词连接成一个句子，并使用空白参数。

那么，我的最终解决方案应该是怎样的呢？

您只需将之前的所有提示连接起来，返回结果句子，并将代码封装成函数表达式。强烈建议对输入和输出数据使用类型提示。

下面是整个解决方案：

function toTitleCase(sentence: string): string {

    return sentence
        .split(' ')
        .map(word => word.charAt(0).toUpperCase() + word.slice(1).toLowerCase())
        .join(' ');
}
