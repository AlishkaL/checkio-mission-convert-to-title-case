使用str.split() 方法（不带参数）将给定句子拆分成单词。您将收到一个单词列表。 

lst = sentence.split()
接下来要做什么？

每个单词都应使用str.capitalize() 方法进行转换。使用map() 函数对每个单词执行上述方法。 

obj = map(str.capitalize, lst)

Map 函数返回一个对象，您可以将其用作其他函数的参数。

我还需要提示。

现在，您需要使用str.join() 方法（带空白参数）将所有大写单词连接成一个句子。 

return " ".join(obj)

您可以不使用额外的变量！

return " ".join(map(str.capitalize, sentence.split()))
那么，我的最终解决方案应该是怎样的呢？

您只需将之前的所有提示连接起来，返回结果句子，然后将代码封装成函数表达式即可。强烈建议对输入和输出数据使用类型提示。

下面是整个解决方案：

def to_title_case(sentence: str) -> str:

    return " ".join(map(str.capitalize, sentence.split()))
