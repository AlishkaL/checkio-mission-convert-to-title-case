How to join all together?

Yes, you need to join all capitalized words back into single sentence with string.join() method with white space argument.

So, how my final solution should look like?

You just need to join all previous hints, return the resulted sentence and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

function toTitleCase(sentence: string): string {

    return sentence
        .split(' ')
        .map(word => word.charAt(0).toUpperCase() + word.slice(1).toLowerCase())
        .join(' ');
}
