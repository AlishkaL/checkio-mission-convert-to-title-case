Catch an idea üí°! Split the given sentence into separate words, capitalize every word (title first character and lower others) and join word back into sentence with white spaces.

Ok, what should I start from?

Split the given sentence into separate words with str.split() method without argument. You receive a list of words. 

lst = sentence.split()
What to do next?

Every word should be converted with str.capitalize() method. Use map() function to implement above method to every word. 

obj = map(str.capitalize, lst)

Map function returns an object and you may use it as an argument in some other function.

I still need a hint.

Now you need to join all capitalized words back into single sentence with str.join() method with white space argument. 

return " ".join(obj)

You may do it without additional variables!

return " ".join(map(str.capitalize, sentence.split()))
So, how my final solution should look like?

You just need to join all previous hints, return the resulted sentence and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def to_title_case(sentence: str) -> str:

    return " ".join(map(str.capitalize, sentence.split()))
